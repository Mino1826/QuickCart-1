# QuickCart Practice Project

QuickCart is a full-stack e-commerce practice project.

This repository is based on a learning project and is used to practice full-stack development concepts such as product management, frontend UI, backend APIs, and database integration.

## Tech Stack

- Next.js
- React
- JavaScript
- Node.js
- MongoDB

## Features

- E-commerce UI
- Product listing
- Cart functionality
- Backend API practice
- Database integration practice

## Important Note

This project is based on a tutorial/forked project and is used for learning purposes.

## What I Learned

- Working with full-stack JavaScript projects
- Understanding Next.js project structure
- Practicing e-commerce features
- Connecting frontend and backend logic
